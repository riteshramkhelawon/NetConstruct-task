# NetConstruct Developer Assessment

To run the application, please load the project into Microsoft Visual Studio and run via IIS Express. 


# All excercises have been completed, with the following features:
- Ability to display dynamic content based on the blog id present in the url (e.g. /blog/1, /blog/2)
- Displaying comments linked to certain blog posts
- Ability to add comments to blogs and save these in the Blog-Posts.json file
- Ability to reply to comments and save these appropriately in the Blog-Posts.json file

## Extras:
- Use of ajax and JQuery for email validation (server-side)
- Use of Bootstrap classes for responsive layout
- Use of a github repo to deliver the assessment
